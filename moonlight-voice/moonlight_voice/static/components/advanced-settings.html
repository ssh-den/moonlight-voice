Trusted-network management only

Home Assistant Ingress is authenticated. The direct management port has no built-in authentication and must not be exposed to the public internet.

Endpoint settings

Routes are fixed, validated service endpoints. Moonlight Voice has no runtime aliases, proxy targets, or outbound request forwarding.

Endpoint Method Purpose Ingress URL (Web UI)
/health GET Service status —
/config GET Active configuration —
/audio GET, POST, DELETE Default audio inventory and management —
/responses GET, POST, PATCH, DELETE Response library management —
/tts GET, POST Return matching or default audio —
/storage DELETE Clear all saved audio data —

These URLs use Home Assistant Ingress and therefore normally include port 8123. They are for the authenticated Web UI, not for the native TTS integration.
